php在现有环境下编译安装zlib等扩展

发表于 2017-05-31 | 分类于 php

安装的时候是在扩展包里面进行编译安装，所以对现有php环境没有影响

首先进源码的的扩展包

用phpize提示报错Cannot find config.m4.

ls一下发现原来config名字改成了config0. cp过来一份。

成功运行编译好环境

找到以前的配置文件,配置编译安装

编辑php.ini添加extension=”zlib.so”
